about spectre-check

Spectre Check runs once before your interview. It enumerates every running process, every visible window, every audio and video device, and every open browser tab. Combinations commonly used by interview copilots are flagged.

severity levels

CRITICALknown copilot, remote-access tool, or hidden window
WARNINGunsigned, cloaked, or unusual install location
CLEANno suspicious signals

what it can't catch

A second device (phone, tablet) running a copilot nearby. Kernel-level tools that hook the detection APIs themselves. Browser tab URLs on Windows (titles only; URLs only on macOS via AppleScript).
